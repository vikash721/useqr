## 1. Keyword Research & Strategy

### 1.1. Target Keyword Buckets

Our keyword strategy is built around targeting users at every stage of their journey. We've expanded the initial keyword list into more detailed buckets to capture a wider range of search intent.

*   **Core Keywords (High Volume, High Competition):** These are the foundational terms that drive the most traffic in the QR code space.
    *   `qr code`
    *   `qr code generator`
    *   `free qr code generator`
    *   `dynamic qr code`
    *   `qr code for business`
    *   `qr code creator`
    *   `create qr code`

*   **Feature-Specific Keywords:** Targeting users looking for specific QR code features.
    *   `qr code with logo`
    *   `custom qr code`
    *   `qr code colors`
    *   `qr code shapes`
    *   `qr code analytics`
    *   `trackable qr codes`
    *   `bulk qr code generator`

*   **Use-Case & Industry Keywords (High Intent):** These long-tail keywords target users with a specific need or in a particular industry.
    *   `qr code for restaurant menu`
    *   `qr code for marketing campaign`
    *   `qr code for event tickets`
    *   `qr code for business cards`
    *   `qr code for product packaging`
    *   `qr code for real estate`
    *   `qr code for classroom`
    *   `vcard qr code`
    *   `pdf qr code`
    *   `wifi qr code`

*   **Question-Based Keywords (Informational Intent):** Excellent for blog content and establishing authority.
    *   `what is a dynamic qr code?`
    *   `how to create a qr code for a website?`
    *   `how do qr codes work?`
    *   `are qr codes safe?`
    *   `how to track qr code scans?`
    *   `best qr code generator for small business`

*   **Branded Keywords:** Capturing users already familiar with the brand.
    *   `useqr`
    *   `useqr login`
    *   `useqr pricing`
    *   `useqr reviews`
    *   `useqr vs [competitor]`
